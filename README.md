# Nixie_Clock
## Arduino-powered Nixie Clock Project
Features:
  * IN-14 Nixies
  * KN155ID1 Drivers
  * Arduino Mega 2560
  * Custom PCB
  * Adufruit DS3231 RTC breakout
  * Three Neuron's HV power supply kit

See the Wiki for a complete breakdown of the build
